👉GuessTheNumber👈

How to play...

✍ In this game you have to guess the correct number between the given limit
✍ New game button will start the game and if you are already in a game then it will reset the score to 0.
✍ After clicking on "New Game" button, you will get a random number between the given limit(Limit will be displayed in a box), now you have to enter the random number in the given input box and click on "Check" button to ensure if it is correct.
✍ If the number you guessed is right then your score will be increased , limit of numbers will be increased and the number of tries u can take will also be increased.
✍ If the number you guessed is wrong then you your tries will decrease and you will be given hint in the hint box.
✍ Once you reach 0 tries then you will lose the game but you can never win this game becuase the limit will keep on increasing as you keep giving right guesses.
